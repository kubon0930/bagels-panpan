# 写真置き場

サイトで使う写真はこのフォルダに置いてください。

- `panpan-logo-circle.jpg` … 丸いネイビーロゴ（仮ロゴ入り。実際のInstagramアイコン画像に上書き可）
- `panpan-storefront.jpg` … 店舗外観写真（未配置。置いてpushすると Hero と Access に自動表示）
- `ogp-placeholder.jpg` … SNSシェア時に表示される画像（1200×630px、仮画像入り）
- 商品写真 … `plain.jpg` のように置いて、data/site.ts の各商品の `image` にパスを設定

詳しい差し替え手順はプロジェクト直下の README.md を参照してください。
